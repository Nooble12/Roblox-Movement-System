# Roblox Movement System
A simple movement system that currently features "wall mantle / climb."

# How to Install (Roblox Studio)
1) Create a local script in the "StarterPlayerScripts" folder.
  2) Paste "MovementSystem.lua" code into the local script.
3) Create a script in the "ServerScriptService" folder.
  4) Paste the "ServerTweenScript.lua" code into the script.
5) Avatar scale settings must be 100% / default Roblox scale settings.
6) Run the game.

# Current Features
1) Wall mantle / climb. Players can climb walls that are too tall to be jumped over / on. Works similarly to a wall mantle in FPS games.
